# Pyqt5-it-essentials7-exam
A very simple program made with PyQt5 that scrapes information from a webpage and creates an IT exam.

Dependencies:
- PyQt5 (version 5.15.9)
- requests (version 2.31.0)
- BeautifulSoup (version 0.0.1)
